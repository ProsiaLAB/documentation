pub fn add_variational_first_order(&mut self, is_test_particle: bool)

Add and initialize a set of first-order variational particles.

Each call to this method adds one VariationalConfig to the simulation and each variational config adds one variational particle for each real particle.

If is_test_particle is true, then only one variational particle will be added If is_test_particle is false, then variational particles will be added for all real particles
