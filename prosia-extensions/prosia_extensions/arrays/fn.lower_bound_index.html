Function lower_bound_index 

Source 
pub fn lower_bound_index(val: f64, array: &[f64]) -> usize
Expand description

Returns the index of the first element in a sorted slice that is greater than or equal to val.

This is equivalent to find_index_ge but clamps the result to valid indices. If val is less than or equal to the first element, returns 0. If val is greater than or equal to the last element, returns array.len().

§Examples

use prosia_extensions::arrays::lower_bound_index;

let arr = [1.0, 3.0, 5.0, 7.0];

// Insert before any elements ≥ 4.0 → index 2
assert_eq!(lower_bound_index(4.0, &arr), 2);

// Value below first → index 0
assert_eq!(lower_bound_index(0.5, &arr), 0);

// Value above last → index len = 4
assert_eq!(lower_bound_index(10.0, &arr), arr.len());
