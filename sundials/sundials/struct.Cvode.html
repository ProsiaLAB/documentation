pub fn save_statistics(&self, fname: &str)

Save CVODE statistics to a file named cvode_stats.txt in a human-readable table format.

§Panics

Panics if the file cannot be opened for writing, which can happen due to permission issues or if the filesystem is read-only. Panics if the CVODE statistics cannot be printed, which can happen if there is an internal error in the CVODE library. Note that this function will overwrite the file cvode_stats.txt if it already exists.
